<h1>.waypoint/</h1>
<br clear="both">

Internal project files for Waypoint itself — branding assets, AI tooling, and project documentation. This directory is not part of the learner's content; it belongs to the framework.

<br>

## Structure

```
.waypoint/
├── branding/           # Official brand assets (logos, icons, banners)
│   ├── banners/        # Full-width wordmark banners (light and dark variants)
│   ├── icons/          # Stand-alone mark icons for various use cases
│   └── logos/          # Primary logo lockups (light and dark variants)
├── ai-skills/          # Claude AI skill definitions for roadmap generation (v3)
├── docs/               # Internal project documentation and reference material
└── ROADMAP.md          # Redirect — see GitHub Issues and Releases
```

## Branding Assets

All assets are provided in both SVG (source) and PNG (raster) format, and in light-background and dark-background variants where applicable.

| Asset | Use |
|-------|-----|
| `banners/` | README footers, document headers |
| `icons/` | Standalone mark — favicons, navbar badges, watermarks |
| `logos/` | Full wordmark lockup — app headers, presentation use |
