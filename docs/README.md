# docs/

A local set of reference documentation, study guides, whitepapers, and saved resources. This is helpful if you want your resources to be portable and be included in the waypoint repo. If you are using external resources then this folder can be ignored.

## Suggested Structure

```
docs/
└── <milestone>/
    └── <module>/
        └── <filename>.<ext>
```

## Example

```
docs/
└── kubernetes/
    └── architecture/
        ├── cka-exam-guide.pdf
        └── kubernetes-architecture-overview.md
```
