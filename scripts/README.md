# scripts/

Automation scripts, setup helpers, and tooling. Anything that makes your learning environment faster to spin up or tear down lives here.

## Suggested Structure

```
scripts/
└── <subject>/
    └── <script-name>.sh or .py or .ps1
```

## Example

```
scripts/
└── kubernetes/
    ├── setup-local-cluster.sh
    └── teardown.sh
```
